Ui-Elements Components

Elementi di design per la creazione di pagine accessibili e responsive

Banner di campagna (Left , Carousel)

  • Tag di supporto: Picture
  • Numero di img supportato: 2
  • Formato immagine di base: 520 x 360
  • Formato immagine smartphone: 743 x 360
  • Tipologia di responsive img: Crop / Standard ('xs')
  • Allineamento testi: Sx
  • Allineamento img: Top - Centrato
  • Classi a supporto: ".campaign-alfa-left", ".campaign-scroll" (abilitazione carosello)
  • Varianti supportate: ".campaign-alfa-right

Pills Showcase - Vetrina (Left)

  • Tag di supporto: Picture
  • Numero di img supportato: 2
  • Formato immagine di base: 310 x 650
  • Formato immagine smartphone: 743 x 285
  • Tipologia di responsive img: Crop
  • Allineamento testi: Sx, Centrato (solo 'xs')
  • Allineamento img: Bottom - Centrato
  • Classi a supporto: ".mode-img-bottom" (>'sm')
  • Varianti supportate: ".mode-img-top" (>'sm'), ".mode-img-middle" (>'sm')
