Button

Base class

I bottoni sono elementi interattivi fondamentali per la navigazione e l'interazione all'interno dell'interfaccia utente. Vengono utilizzati per eseguire azioni, inviare dati o guidare l'utente verso specifiche sezioni di un'applicazione o di un sito web.

Il kit mantiene la nomenclatura di Bootstrap attraverso la classe .btn, che fornisce lo stile di base per elementi HTML come <button> e <a>.
Questa classe permette di applicare rapidamente uno stile uniforme e funzionale a bottoni e link, garantendo un aspetto uniforme e funzionale nella UI.
