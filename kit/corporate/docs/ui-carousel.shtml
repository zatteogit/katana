Carousel

Carousel - Swiper Base

Il Carosello è un componente interattivo che consente di mostrare una serie di contenuti, come immagini o card, in uno spazio limitato, permettendo agli utenti di navigare tra di essi tramite controlli intuitivi.

Il kit-Html fa uso di Swiper.js, un potente framework per slider per la realizzazione di caroselli responsive ed accessibili.

Dipendenze

  • swiper-bundle.min.js
  • swiper-bundle.min.css
  • swiper-corporate.css

Markup

La struttura include un container principale .swiper-container con l'attributo aria-roledescription="carousel", indicando agli screen reader che si tratta di un carosello.
Le slide sono gestite all'interno di un macro contenitore swiper-wrapper. Ciascun pannello è all'interno di uno swiper-slide

La classe .bg-track permette di visualizzare il "binario" (di default in grigio) posto sullo sfondo del carosello; la classe .bg-track-over permette di sovrascrivere il posizionamento di default del binario.

La corretta visualizzazione del carosello è strettamente correlata all'utilizzo di attributi data personalizzati elencati qui di seguito con i possibili valori attribuibili:

Carousel - Swiper Base DATA attribute
DATA attribute Valori Descrizione
data-pi-position default / track / track-over gestisce le spaziature e il posizionamento del carosello rispetto alla presenza del Binario.
Javascript

La configurazione di Swiper è impostata tramite args, includendo opzioni come slidesPerView, spaceBetween, pagination, autoplay, navigation e breakpoints per la responsività.
I parametri a11y aggiungono messaggi personalizzati per l'accessibilità in italiano, come indicazioni per le slide successive e precedenti e l'indicazione di numero slide.

Carousel - Swiper Gallery

Il componente Swiper Gallery è una galleria di immagini con supporto per miniature di navigazione. Utilizza Swiper.js per fornire un'interfaccia scorrevole e interattiva, con controlli di navigazione e paginazione.

Dipendenze

  • swiper-bundle.min.js
  • swiper-bundle.min.css
  • swiper-corporate.css

Markup

La gallery è composta da diversi elementi che lavorano insieme per offrire un'esperienza di navigazione fluida e intuitiva.

Il componente principale è lo swiper principale .swiper-preview, che contiene le immagini visualizzate a grandezza intera.

A supporto, per una selezione più rapida, è presente lo swiper delle miniature .swiper-thumbs, visibile solo su dispositivi con viewport superiori a MD.

Per facilitare la navigazione tra le immagini, sono disponibili i pulsanti di navigazione .swiper-btn-prev e .swiper-btn-next, che consentono di scorrere avanti e indietro nel carosello.

Infine, la paginazione .swiper-pagination fornisce un'indicazione visiva della posizione corrente all'interno della galleria.

Javascript

La configurazione di Swiper è impostata tramite args, includendo opzioni come slidesPerView, spaceBetween, pagination, autoplay, navigation e breakpoints per la responsività.
I parametri a11y aggiungono messaggi personalizzati per l'accessibilità in italiano, come indicazioni per le slide successive e precedenti e l'indicazione di numero slide.
