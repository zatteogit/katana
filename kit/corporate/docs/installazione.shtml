Installazione

Distribuzione Kit HTML

Per la condivisione del pacchetto kit HTML sono stati definiti specifici passi operativi definiti all'interno di COO/SI/Governance (ora DTO) al fine di rendere fruibile alle strutture di sviluppo interessate la componente User Interface KIT (oggi kit HTML).

Il gruppo CA/DM/Progettazione Grafica si occupa del caricamento del pacchetto compresso all'interno della cartella dedicata, denominata "UI-kit HTML", su ambiente SharePoint.

Il sistema SharePoint invia automaticamente una notifica via mail ad un elenco predefinito di utenti (da aggiornare a cura dei PQM delle strutture di Sviluppo) ogni qualvolta viene rilasciata una nuova versione del kit HTML. I PM/PQM hanno accesso in lettura alla cartella per acquisire il kit ed utilizzarlo, ove applicabile.

Della richiesta di accesso alla sopra citata cartella si fa carico la struttura di DTO stessa.

NB: Per la visualizzazione della linea guida è necessario decomprimere l'archivio su una doc_root di un web server / virtual host con SSI abilitati.
