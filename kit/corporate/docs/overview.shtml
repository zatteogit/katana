Overview del pacchetto

Intro

Il Kit, realizzato in HTML, CSS e JavaScript, fornisce esempi e linee guida utili per garantire la coerenza grafica nei vari touchpoint digitali web di Poste Italiane, in linea con le direttive di design e progettazione.

La guida è sia uno strumento, sia un investimento per mantenere un vantaggio competitivo nel panorama digital.

 

Dettagli tecnici

La sua realizzazione, o più nello specifico degli strumenti e del codice sorgente che ne fan parte, è basata sui princípi di composizione e riuso di componenti condivisi.

Realizzato su base Bootstrap 5 garantisce compatibilità con i browser moderni, ottimizzazione visiva, facilità di utilizzo e riduzione dei tempi di sviluppo. Fra le risorse del kit sono inoltre a disposizione pagine HTML di esempio da usare come punto di partenza per lo sviluppo del tuo progetto.

La griglia di oggetti fa uso delle proprietà native css Flexbox e Grid che rendono i layout più flessibili e reattivi.

L'introduzione di Sass ha permesso di creare un sistema di stili modulare e scalabile migliorando le performance generali.

Ogni esempio è stato predisposto avendo come riferimento le linee guida per l’accessibilità dei contenuti web WCAG 2.1.

Ovviamente il Kit HTML è in continuo e costante aggiornamento. Tali aggiornamenti supportano l'adeguamento alla User Experience ed User Interface codificata e la risoluzione di eventuali anomalie.

 

Contenuti organizzati

Una volta estratto il contenuto del pacchetto "tar.gz" saranno visibili le cartelle principali e una "index" che permette di esplorare i contenuti del kit HTML a partire dall'area documentazione.

Macro struttura Directory
Directory Descrizione contenuto
dist/* Contiene il kit Html vero e proprio comprensivo di risorse javascript e fogli di stile
dist/assets/core/* Contiene le risorse proprietarie del kit
dist/assets/vendor/* Contiene le risorse di terze parti, come ad esempio il framework di riferimento Bootstrap
docs/* Contiene la documentazione del pacchetto.
 

Codice e preview

La documentazione del kit HTML permette di avere, per ogni singolo asset, una preview dell'oggetto realizzato e la trasposizione del codice stesso.

La scheda del singolo asset permette di individuare velocemente tutte le caratteristiche dell'oggetto presente in kit con varianti (correlate all'utilizzo di <tag/> o .classi specifiche), relative dipendenze e/o riferimenti a pagine già realizzate dove trovare un esempio di applicazione.

La funzionalità di preview genera un'anteprima dell'asset simulandone il comportamento su diversi cutpoint.

Seguono, a titolo di esempio, un blocco dipendenze, tre blocchi di codice, una preview di pagina, una porzione descrittiva e un blocco lista link a pagine esemplificative.
