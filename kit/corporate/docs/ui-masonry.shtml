Masonry

Masonry Grid

L'asset "Masonry" è un layout di griglia utilizzato per visualizzare immagini o altri contenuti in uno stile simile a un "muro di mattoni". Gli elementi sono disposti in colonne e possono avere altezze variabili, creando un effetto visivo interessante e dinamico. Questo layout è ideale per gallerie di immagini, portfolio fotografici o qualsiasi contenuto visivo che non richieda una disposizione uniforme degli elementi.

E' possibile utilizzare questo layout predisponendo la classe .masonry sul contenitore degli asset che si vuole gestire in griglia.

Per garantire il corretto funzionamento del Masonry Grid è necessario caricare entrambe le dipendenze js.

  • La libreria masonry.pkgd.min.js permette il funzionamento generale.
  • Con l’utilizzo di imagesloaded.pkgd.min.js, il layout si aggiorna ogni volta che un'immagine viene caricata (o che ci sia un cambio di cutpoint), riducendo eventuali problemi di sovrapposizione.

percentPosition: true consente a Masonry di calcolare le posizioni degli elementi in base alla percentuale.
