Icons

Icons - Sprite

Tutte le icone sono organizzate in una singola cartella. Al suo interno si trova uno sprite che raccoglie tutte le icone, insieme alle versioni individuali di ciascuna di esse, in tre colori base: bianco, blu e nero. La nomenclatura delle icone varia a seconda del colore, con l'aggiunta di un suffisso che indica il colore specifico (ad esempio, *-neutral100 per il bianco, *-primary33 per il blu).

Macro struttura Directory - Icons
Directory Descrizione contenuto
/dist/assets/core/images/iconset/* Contiene tutte le risorse grafiche (icone) in svg

In base alle esigenze del progetto, è possibile scegliere di utilizzare lo sprite per caricare tutte le icone in un'unica risorsa, ottimizzando le performance, oppure selezionare le singole icone in base al colore desiderato, per un controllo più preciso e mirato. L'uso dello sprite è particolarmente utile per gestire dinamicamente i colori delle icone SVG tramite il CSS, evitando di dover caricare più versioni della stessa icona. Questa flessibilità consente di adattarsi a diversi scenari di implementazione, garantendo sia efficienza che personalizzazione.
