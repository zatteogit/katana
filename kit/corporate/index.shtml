Html Kit - 1.0

Corporate • Linea guida grafica

  • Contenuto

    Il Kit è una guida in HTML/CSS/Javascript che ha l'obiettivo di aiutare a mantenere la coerenza grafica sui vari touch point digital web di Poste Italiane, in linea con il design e la progettazione.
    Basato su framework Bootstrap 5.x.

  • Installazione

    Estrai il contenuto del pacchetto all'interno della document root del webserver HTTP con "Server Side Include (SSI)" abilitati per visualizzare correttamente il contenuto.
    Per un utilizzo "in locale" è possibile usufruire di un qualsiasi software in grado di creare server web di tipo HTTP (suggeriti MaMP, WaMP, XaMPP).
